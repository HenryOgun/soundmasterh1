# Henry Ogun - Professional Blog

A clean, professional blog website for Henry Ogun (SoundMasterH1) - Broadcast Engineer and Full-Stack Developer.

## Features

- **Professional Design**: Clean, modern interface with excellent UX
- **Responsive Layout**: Works perfectly on all devices
- **Interactive Navigation**: Smooth section navigation
- **Project Showcase**: Featured projects in broadcast engineering and development
- **Achievement Timeline**: Professional milestones and awards
- **Contact Integration**: WhatsApp integration for direct messaging
- **Blog Section**: Latest articles and insights

## Technologies Used

- **React 18**: Modern React with hooks
- **Vite**: Fast build tool and development server
- **Tailwind CSS**: Utility-first CSS framework for styling
- **Lucide React**: Beautiful, customizable icons
- **PostCSS**: CSS processing

## Sections

1. **Hero**: Professional introduction with experience badge
2. **About**: Journey and expertise overview
3. **Projects**: Featured work in broadcast and web development
4. **Achievements**: Timeline of professional milestones
5. **Blog**: Latest articles and insights
6. **Contact**: Professional contact form with WhatsApp integration

## Getting Started

1. **Install dependencies**:
   ```bash
   cd henryogun-professional-blog
   npm install
   ```

2. **Start development server**:
   ```bash
   npm run dev
   ```
   Server will start on http://localhost:3002

3. **Build for production**:
   ```bash
   npm run build
   ```

## Customization

- **Profile Image**: Add your h7.jpg photo to `public/images/`
- **Content**: Update projects, achievements, and blog posts in `HenryOgunBlog.jsx`
- **Colors**: Modify the color scheme in `tailwind.config.js`
- **Contact Info**: Update email and phone number in the contact section
